Type Error: Network request failed

重新遇到ios的fetch API报错，Type Error: Network request failed问题
这一次我还是按照上次的有过的经验改了一下，可是一开始用文本编辑器修改了appName/ios/appName/Info.plist还是改了之后没生效，我试过将模拟器重启，还有无数次reload，还是没生效，甚至我还试过长时间等待，再重新reboot模拟器，还是不行

最后使用Xcode直接打开Info.plist来修改，在对应的AppTransportSecurity中点击一个+号，会直接添加一个子key，找到key的名字为Allow Arbitrary Loads就可以了。
