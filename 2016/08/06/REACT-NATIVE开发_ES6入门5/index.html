REACT-NATIVE开发_ES入门5

2016-08-06 • 分类: react • 标签: react-native

为了加快进度，我要忽略很多内容，现在直接跳到函数这一部分来学习了，因为在初学reactnative的时候就遇到了箭头函数。

基本用法

在ES6中允许使用箭头(=>)来定义函数。

小结一下：对比可以看出一点端倪，首先是分析结构：var 变量名字 = 参数列表 => 返回的数值;。也就是“=>”可以替代function这个词语使用并且前后省略了return这个关键字，方便了许多。

另外如果箭头函数不需要参数或者是需要多个参数的时候，就可以使用一个圆括号代表参数部分。

当然如果箭头函数的代码块多于一条语句，就需要用大括号括起来，并且使用return语句返回。由此我可以联想到假如在return之前还需要做一些事情的话，有了这个功能就很好办了。
